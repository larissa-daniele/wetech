# 💻 WETECH

WETECH é um site de tecnologia fictício desenvolvido para um processo seletivo 
de Analista de SEO, com foco na aplicação de técnicas de otimização para 
mecanismos de busca.

## 🚀 Sobre o projeto

O site foi construído com HTML e CSS com o objetivo de praticar a implementação 
de boas práticas de SEO diretamente no código, como estratégia para aumentar 
a visibilidade em ferramentas de pesquisa. Entre as práticas aplicadas estão:

- Uso correto de meta tags e palavras-chave
- Descrição semântica de imagens (atributo `alt`)
- Estrutura de headings organizada (H1, H2, H3)
- Boas práticas gerais de HTML semântico

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- SEO On-Page
